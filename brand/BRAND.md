# π-slice Brand Assets

**Aesthetic:** Kitschy retro bakery — fresh-baked pie meets mathematical constant.

## Files

| File | Use |
|---|---|
| `logo-full.svg` | Primary logo with plate, tablecloth, steam. README hero, docs. |
| `icon.svg` | 128×128 mark. Favicons, app icons, small contexts. |
| `badge.svg` | GitHub-style shield badge for README. |
| `social-banner.svg` | 1280×640 OG/social preview image. |
| `wordmark-light.svg` | Text-only logo for light backgrounds. |
| `wordmark-dark.svg` | Text-only logo for dark backgrounds. |
| `powered-by.svg` | Footer attribution badge. |

## Color Palette

| Name | Hex | Use |
|---|---|---|
| Crust Gold | `#E8A849` | Primary brand, highlights |
| Crust Dark | `#A05A1B` | Depth, borders |
| Cherry Filling | `#D4423B` | Accent, the π character, CTAs |
| Filling Dark | `#8B1A15` | Deep accent |
| Baker's Brown | `#4A2508` | Text, dark backgrounds |
| Cream | `#FDF6EC` | Light backgrounds, text on dark |
| Tablecloth Red | `#E85D4A` | Decorative pattern |

## Typography

- **Display/Logo:** Georgia or Times New Roman (serif) — the bakery vibe
- **Code/Mono:** Courier New — for terminal/install contexts
- The π character is always `#D4423B` (Cherry Filling)
